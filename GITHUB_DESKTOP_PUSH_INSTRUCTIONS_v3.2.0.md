# GitHub Desktop Push — v3.2.0

## Steps

1. Open **GitHub Desktop** on your Mac
2. Make sure the repository **football-prediction-game** is selected
3. You will see the following modified files in the left panel:
   - `app.js`
   - `index.html`
   - `styles.css`
   - `README.md`
   - `docs/index.html`
   - `docs/APPLICATION_DEFINITION.md`
   - `docs/TECHNICAL_SPECIFICATION.md`
   - `docs/SOURCE_CODE_DOCUMENTATION.md`
   - `RELEASE_NOTES_v3.2.0.md`
   - `GITHUB_DESKTOP_PUSH_INSTRUCTIONS_v3.2.0.md`
   - New backup folder under `backups/`

4. In the **Summary** field at the bottom left, enter:
   ```
   v3.2.0 — Knockout 3-stage predictions (90min/ET/Penalties) + Bob AI + Admin result entry
   ```

5. Click **Commit to main**

6. Click **Push origin** (top right)

**Done!** All users will receive the update automatically on their next page load.
