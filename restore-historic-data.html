⚠️ What Happened?

Versions v2.0.0 through v2.0.6 had a bug where match data (results and Bob's predictions) was not migrated from localStorage to Firebase. When the migration cleared localStorage, this data was lost.

What was lost: Match results, Bob's predictions, match history

What was preserved: User accounts, user predictions, pools, coins
